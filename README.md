
# internship task

for code reference check src/index.ts



## Run Locally

Clone the project

```bash
  git clone https://github.com/Meditatingpanda/internship-task.git
```

Go to the project directory

```bash
  cd internship-task
```

Install dependencies

```bash
  npm install
```

run the unit tests

```bash
  npm run test
```
